Projetos

Descubra nossos projetos que refletem nossa paixão pela inovação e design. Cada trabalho é uma combinação de criatividade e funcionalidade, criado para atender às necessidades únicas de nossos clientes.

Site de Estagio

Desenvolvi um site focado em facilitar o acompanhamento e o cadastro de estágios acadêmicos. Neste projeto, implementei funcionalidades que permitem o monitoramento das atividades de estágio, auxiliando tanto alunos quanto administradores no gerenciamento das informações. Utilizei tecnologias como Node.js para o backend, MySQL 8.0 para o banco de dados, além de HTML, CSS e JavaScript para a interface, proporcionando uma experiência fluida e intuitiva para os usuários.
